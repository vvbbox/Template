TutoCito CMS

TutoCito CMS — система управления контентом веб-сайтов, мощное веб-приложение SPA (Single Page Application), разработанное на платформе ASP.NET CORE, позволяет создавать и управлять контентом сайта в режиме реального времени, надежно и безопасно сохранять этот контент. Адаптирована для использования на мобильных устройствах. Идеальна для личных, бизнес и коммерческих проектов.


  • Одностраничное приложение (SPA) для быстрой и удобной работы
  • Редактор WYSIWYG для удобного редактирования
  • SEO-оптимизация и расширенное управление тегами
  • Гибкое управление доступом и поддержка нескольких баз данных
  • Открытый исходный код по лицензии MIT

Основные возможности

  • Управление контентом в реальном времени
  • Ролевое управление доступом
  • REST API для интеграций
  • Настраиваемые HTML-шаблоны страниц
  • Мобильная адаптация и отзывчивый дизайн
  • Простая установка и настройка на VPS
  • Профессиональная поддержка
